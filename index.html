Welcome to Turtle Dialer for Chrome

About

Turtle Dialer for Chrome lets you make phone callS from your browser.

You can call a phone number found on a web page by simply selecting the number and peeking the option Call to that appears on the context menu.

It also adds an action button to the right of the Chrome's address bar that opens a popup that contains a phone book and a call log.

Documentation

Install

Get Turtle Dialer at Chrome Web Store.

Configure

In the address bar, type chrome://extensions and choose options.

In the settings page, select the country where you are based, the IP phone model and the checkbox if you want to include your Gmail contacts in the Turtle Dialer's phone book.

To allow Turtle Dialer to make remote calls, you will have to authorize the IP address of your PC.

Please choose between the following two options depending on the model of the phone from which you are going to make the call.

Yealink T20P

Connect to your phone's administration console, go to Phone -> Features -> API Security and type a list of ip addresses in the field Trusted Action URI Server List or enter any if you want to allow any ip address, for example in case your PC has an automatic assigned ip (dhcp).

Yealink T28P

Connect to your phone's administration console, go to Features -> Remote Control and type a list of ip addresses in the field Action URI allow IP List or enter any if you want to allow any ip address, for example in case your PC has an automatic assigned ip (dhcp).

The first time you make a call from Turtle Dialer a message appear on the phone display asking you to authorize the remote control.
